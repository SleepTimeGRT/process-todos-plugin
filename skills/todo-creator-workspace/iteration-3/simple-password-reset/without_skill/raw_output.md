```03-implement-reset-link-email-api.md
# 비밀번호 리셋 이메일 발송 API 구현

이메일 주소를 받아 해당 계정에 비밀번호 재설정 링크를 발송하는 백엔드 엔드포인트를 구현한다. 토큰을 생성하고 DB에 저장한 뒤 이메일로 전송한다.

- [ ] `POST /api/auth/forgot-password` 엔드포인트 생성
- [ ] 요청 이메일로 사용자 존재 여부 조회
- [ ] 고유한 재설정 토큰 생성 (crypto 등 사용) 및 만료 시간 설정
- [ ] 토큰을 DB에 저장 (user_id, token, expires_at)
- [ ] 이메일 발송 서비스 연동하여 리셋 링크 포함 메일 전송
- [ ] 존재하지 않는 이메일 요청 시 보안상 동일한 성공 응답 반환 (이메일 열거 방지)
- [ ] 이메일 발송 실패 시 에러 핸들링 및 로깅
```
